# SpellCheckerProject
This is a spell checker program written in C++ that reads a dictionary containing correct words all in lowercase,
reads an input text file and provides suggestions to incorrect words using edit distance and output the corrected text.

# Setup 
1. Open in Visual Studio Code
2. Install the C/C++ extension on Vs code

# Running the code
 - Press `Ctrl+Shift+B` (or `Cmd+Shift+B` on Mac) to run the default task
 - Or press `F5` to start debugging

# Features
- Edit Distance( Dynamic Programming )
- Best 3 suggestions for incorrect words
- Use of Unordered_Set for fast lookup of correctly spelt words
- Text is entered from an input file
- Outputs the results to an output file
- Suggests a possible correct sentence

# Instructions
1. Enter the sentence that you would like to be checked into the input file
2. Click on the main file and run it
3. A confirmatory text will be displayed at the terminal once run succesfuly
4. The result is displayed at the output file
